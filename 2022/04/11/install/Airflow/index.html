[Windows 10] Setting up Apache-Airflow in Windows using WSL2🐱‍🏍

✨설치 전, WSL2 설치 필수.✨

01. Install pip on WSL

  1. airflow 설치하기 위해 pip 설치
  2. sudo apt install python3-pip

02. Install virtualenv package

  1. virtualenv 라이브러리 설치
  2. sudo pip3 install virtualenv

03. Create a virtual environment

  1. C 드라이브에 airflow-test 폴더 생성
  2. mkdir 입력하고 생성 폴더 이름 적기
  • ex. mkdir airflow-test 이렇게 적으면 현재 내 위치에 폴더가 생성된다.(내 위치 : mnt/c:)
  1. 가상 환경 생성
  • virtualenv venv
  1. 가상 환경 접속
  • source venv/bin/activate
  1. 환경 변수 열고 코드 추가
  • vi ~/.bashrc ——→ export AIRFLOW_HOME=/mnt/c/airflow-test
  • 환경 변수를 열고 코드 추가하기 위해서는 esc + i 눌러 작성.
  • 코드를 맞게 적었다면, :wq! ———→ 단축키 : 저장하고 나가기.
  • 저장 안하고 나간다면 그냥 q! 적기.
  1. 수정 된 코드를 업데이트 하기 위한 코드
  • source ~/.bashrc
  1. 실제 코드가 반영되었는지 확인 하기 위한 코드
  • echo $AIRFLOW_HOME —(결과)—> /mnt/c/airflow-test

04. Apache Airflow 설치

  1. PostgreSQL, Slack, Celery 패키지를 동시에 설치하는 코드를 작성
  • pip3 install 'apache-airflow[postgres, slack, celery]'
  1. airflow 설치 위해 DB 초기화 해줘야 한다.
  • airflow db init
  1. 실제로 잘 구현이 되었는지 확인하기 위해 webserver 실행
  • airflow webserver -p 8081
  • 뒤에 번호는 때에 따라서 달라질 수 있음.
  1. 링크에 접속하여 아래와 같은 화면이 나타나지는 확인 한다.

  • http://localhost:8081/login/

  1. username 생성을 하지 않았다. 아래와 같이 추가 하는 코드 작성.
  • airflow users create --username airflow --password airflow --firstname evan --lastname airflow --role Admin --email your_email@some.com
  • 각 해당 목록 마다 본인 걸로 지정.
  1. 다시 링크 접속 후 생성된 username, password 적고 접속. 아래 와 같이 뜬다면 성공!
