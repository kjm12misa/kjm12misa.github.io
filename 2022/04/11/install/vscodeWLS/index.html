[Windows 10] VSCode WLS 연동✨

✨설치 전, airflow 설치 후 진행.✨

  1. VSCode 설치

  • • URL : https://code.visualstudio.com/download

  • 관리자로 실행할 것이기 때문에 System Installer 다운로드.

  • 본인 컴퓨터 사양 확인 후 다운로드.

  1. 설치 시, 환경 변수 체크란 잘 확인 할 것.

Remote WSL 연동

  1. visual studio code 관리자 모드로 실행

  2. Extension 버튼 클릭 → Remote WSL 검색하고 설치 → 모두 체크 후, Mark Done 선택.

  3. open folder 클릭 → WSL에서 C 드라이브(저장 되어 있는 드라이브)에 저장 했던 airflow-test 폴더 선택.

  4. 해당 프로젝트가 화면 왼쪽에 열려 있는 것을 확인.

  5. 메뉴 바에 Terminal 선택. → New Terminal 선택.

  6. 생성된 터미널에 WSL이 있는지 확인.

  7. 생성된 WSL 에 필요한 데이터 다운로드.(p. 45)

  • pip3 install faker, pip3 install pandas

간단하게 python 파일 실행.

  1. AIRFLOW-TEST 클릭 → 새로운 폴더(chapter03) → 새로운 파일(step01_wirtecsv.py)생성

  2. 생성된 파일 열고 print(“dddd”) 적고 ctrl + s 눌러서 저장

  • 반드시!!! 작성하고 나서 저장을 해야지 터미널에서 출력이 된다.
  1. 그림과 같이 순서대로 세팅 후 진행.

  1. python3 step01_writecsv.py 입력.
  • 출력을 할 경우 앞에 python3를 붙여줘야 하며, 파일이 많을 경우 특정 앞 문자만 적고 TAB 버튼을 눌러 자동 완성 시키자. step01_(TAB)writecsv.py
  • TAB 눌러 주는 건 내가 제대로 파일을 잘 생성한 것 인지에 대한 확인도 가능.

  1. 출력 된 값 확인.
