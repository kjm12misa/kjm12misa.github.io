[Windows 10] GitHub blog 만들기✨

Step 01_설치

  • Git bash, node .js, vs code 설치를 전제로 진행
  • 마우스 우 클릭 git bash 열기
  • hexo init 깃허브 생성해 놓은 이름(ex. hexo init myblog)
  • Git bash 에서 생성 된 myblog 폴더가 있는 곳으로 경로 설정
  • code . 입력하여 vs code 실행

Step 02_git blog 생성

  • github에 뜬 방법 복사 붙이기
  • git add . (모든 파일 올리기.) or git add .github_new(파일이름)
  • git commit -m “설정 이름(영어로, ex. ”updated”)
  • git push (업로드)
  • GitHub 사이트에서 새로 고침 하여 올라간 파일 확인.
  • 3가지 명령어를 순차적으로 install(맨 앞에 공백 없이)
  • GitHub에 새로운 아이디 생성하기 위해 config.yml 에서 수정.
  • url: [https://kjm12misa.github.io](https://kjm12misa.github.io) 수정

  • 아래와 같이 설정 해 줄 것
  • type: git
    repo: https://github.com/kjm12misa/kjm12misa.github.io.git
    branch: main

  • 아래와 같이 설정이 된 것을 확인

  • “branch : main” 을 잘못 적거나 정해 놓지 않은 이름을 적으면 오류가 난다.
    다른 이름으로도 설정이 가능하다.

Step 03_GitHub 파일 업로드

  • hexo generate —deploy(파일 업로드) or hexo generate → hexo server(서버 확인)

Step 04_source-posts 파일 생성

  • hexo new “영어로 파일 이름 작성“
  • Ex) hexo new "My New Post"

Step 05_기타

  • hexo clean 해주기. (쿠키 등 없애줌)

  • 참고 사이트 : https://dschloe.github.io/settings/hexo_blog/

  • github 배포 문제점 확인 : https://www.githubstatus.com/
