[Windows 10] WSL2 설치 및 사용법✨

✨첫 번째로 설치.✨

필요 항목 설치 및 준비

  1. pc 정보 → windows 사양 → 버전 확인(20H1, 20H2, 21H1 ..)
  2. 버전이 낮으면 windows update 설정을 열어 최신 버전으로 업데이트.
  3. WSL2를 설치 하기 위해서는 가상 터미널을 사용해야 한다. Windows Terminal 사용 추천.
    powershell 사용해도 되지만 windows Terminal 에서 WSL2를 바로 실행하는 것도 가능하기 때문에 미리 설치해두는 것 추천!

ubuntu 및 WSL2 Linux 설치

  1. Windows Terminal 관리자로 실행
  2. dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart 입력. → 아래와 같이 출력.

  1. dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart 입력. → 아래와 같이 출력.
  • 작업을 완료했습니다. 메시지를 꼭 확인할 것. 작업이 정상적으로 완료 되었다면, 재부팅 한 번 하기.

WSl2 Linux 커널 업데이트

  • https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi (x64 머신용 최신 WSL2 Linux 커널 업데이트 패키지) 다운로드.
  • windows Terminal에 다음 명령어 입력. 기본적으로 사용할 WSL 버전을 2로 변경 해줍니다.
  • wsl –set-default-version 2

마이크로소프트 스토어에서 리눅스 설치

  1. 관리자 모드로 실행 후 처음 ‘Installing. this may take few minutes…’ 메시지가 나오고, 시간이 지나면 사용자 이름과 패스워드를 지정하는 입력 창이 나타난다. 사용하고자 하는 사용자 이름 , 패스워드 지정.

이 때, 메시지가 뜨지 않는다면 !!!

  • 윈도우에 검색 → windows 기능 켜기/끄기에서 체크 해제 후 재 부팅 → 다시 체크 확인 후 재 부팅 → 우분투 다시 관리자 모드로 실행 후 메시지가 뜬다면 성공!

  1. wsl -l -v 입력
  • 현재 설치된 리눅스 버전 확인
  • 버전에 2가 출력이 된다면 그대로 진행. 1이 뜬다면 2번을 입력.

  1. wsl –set-version Ubuntu 2
  • 아래와 같이 ‘변환이 완료 되었습니다.’ 라는 메시지가 뜰 때 까지 기다린다.
