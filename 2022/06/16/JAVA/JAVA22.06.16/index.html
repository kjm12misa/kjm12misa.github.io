🎈JAVA_변수(variable)

이것이 자바다.

1. 변수

  • 변수란, 하나의 값을 저장할 수 있는 메모리 공간.

1_1. 작성 규칙

규칙 예시
첫 글자로는 숫자X, 문자 또는 ‘$’, ‘_’로 지정 가능(필수) O : price, $price, _price
X : 1price, @price, $#price
영어 대소문자 구분(필수) firstname과 firstName은 다른 변수
첫 문자는 영어 소문자로 시작, 다른 단어가 붙는 경우 첫 문자는 대문자(관례) maxPrice, minPrice, firstName
문자 수(길이)의 제한 없음.
자바 예약어는 사용X 이유 : 예약어들로 변수 이름 지정 시 컴파일 에러 발생

1_2. 자바 예약어

  • 기본 데이터 타입
    • 정수 타입 (byte, short, int, long)
    • 실수 타입 (float, double)
    • 문자 타입 (char)
    • 논리 타입 (boolean)
      • 자료형, 타입, 메모리 사용 크기 정도는 기억할 것.

  • 접근 지정자

    • public : 모든 클래스의 접근 허용.

    • protected : 상속 받은 클래스 또는 같은 패키지에서만 접근 가능

    • private : 외부에서 접근 X, 같은 클래스 내에서만 접근 가능

      • 예시

    • defult : 아무것도 붙이지 않아도 되고, 자신의 클래스 내부와 같은 패키지 내에서만 접근 가능

      • 예시

      }

      • 패키지란, 관련된 클래스 혹은 인터페이스들을 묶어 놓은 상자와 같음.

    • 접근 지정자(접근 제한자) 사용
    • 클래스 : public, default
    • 생성자 : public, protected, default, private
    • 멤버 변수 : public, protected, default, private
    • 멤버 메소드 : public, protected, default, private
    • 지역변수 : 접근 제한자 사용 불허

  • 클래스와 관련된 것

    • class, abstract(추상), interface, extends, implements, enum

  • 객체 관련

    • new, instanceof, this(자신), super(부모), null

  • 메소드 관련

    • void, return

  • 제어문 관련

    • if, else

    • switch : 어떤 변수의 값에 따라서 문장을 실행할 수 있도록 하는 제어문

      • 사용 키워드 : switch, case, default, break

    • case, default, for(반복문), do, while, break, continue

  • 논리 값(타입으로는 boolean만 가능)

    • true, false

  • 예외 처리와 관련

    • try, catch, finally, throw, throws

  • 기타

    • transient, volatile, package, import, synchronized, native, final, static, strictfp, assert
