8장 통계 분석

  • 유의수준은 .05, .01, .001 등 세종류가 있다. 유의수준이 낮을수록 오류 허용 범위가 적고, 신뢰성을 높여 줍니다.

(1) 기술 통계(descriptive statistics)

  • 데이터의 특징을 서술한다 또는 사실 확인
  • 평균, 최소값, 최대값, 중앙값

(2) 추론 통계(inferential statistics)_<

  • 변수 간의 관계 파악, 인과 관계, 새로운 사실 발견.
  1. 평균 차이 검정 : 집단별로 평군의 차이가 실제 있는지 확인
  • 분석 사례_t.test() 함수 사용
  • 방법 1 : t.test(data=데이터세트, 종속변수(비교값)~독립변수(비교대상))
  • 방법 2 : t.test(data=데이터세트$종속변수(비교값)~데이터세트$독립변수(비교대상))

  • p-value = 1.263e-05 유의수준이 1.263e-05

  • e-n : 10의 n승 분의 1, e+n : 10의 n승 의미

  • 1.263e-05 -> 1.263/100000

  • 유의수준 허용 조건(p < .05) 보다 적기 때문에 조건 충족

  • alternative hypothesis: true difference in means between group Man and group Woman is not equal to 0

  • 대립가설 : 평균 차이가 있다.

  • 95 percent confidence interval: 신뢰수준 95%

  • -3.887311 -1.527033

  • auto : 15.96.., manual : 18.67.. 두 값의 평균 차이가 있다는 대립 가설을 채택할 수 있음.

결론

  • cty 평균거리는 auto 15.97마일, manual 18.68마일. 유의수준의 허용 조건인 0.05보다 작아 통계적으로 유의미한 차이기 있어 auto 보다 manual가 2.7마일 길다고 볼 수 있다. (확률을 토대로 한 추론분석이라서 ‘볼 수 있다’ 표현.)

  1. 상관관계분석
  • 변수 간의 연관성 즉, 한 변수가 변화하면 다른 변수도 변화 하는 관계
  • 귀무가설(H0) : cty, hwy는 상관관계가 없다.
  • 대립가설(H1) : cty, hwy는 상관관계가 있다.
  • cor.test() 함수 사용. # cor.test(데이터세트$비교변수1, 데이터세트$비교변수2)

(1) 단순회귀분석

  • 독립변수 1개, 종속변수 1개일 때 쓰임.
  • lm() 함수 사용 -> lm(data=데이터세트, 종속변수독립변수), lm(데이터세트$종속데이터세트$독립)

help(mtcars)

값 분석

  • disp 계수 값은 -0.04122, 절편은 29.59985
  • 회귀분석에서 구해진 식 : mpg = 29.59985 - 0.04122 * disp
  • 배기량이 1단위 씩 올라 갈때마다 mpg는 0.04122 씩 감소
  • 회귀분석은 유의수준이 없으면 회귀계수는 의미가 없으므로 summary(회귀분석식)를 사용.

값 분석

  • p-value : 9.38e-10(9.38/10^10 -> 9.38/10000000000 =0.000000000938)허용치 안에 있어 회귀모형 적합
  • 대립가설만 채택 가능.
  • Signif. codes: 0 ‘’ 0.001 ‘’ 0.01 ‘’ 0.05 ‘.’ 0.1 ‘ ’ 1 : 세가지 유의수준 유형을 모두 충족함. (유의수준기호)
  • Adjusted R-squared: 0.709 -> 수정된 결정계수(R^2): 0.709
  • 결론 : 회귀모형은 유의수준에 적합하며, 수정된 결정계수는 .709 이다. 배기량이 연비에 미치는 회귀계수는 유의수준 p < .001에서 -0.04이다.

(2) 다중회귀분석

  • 종속변수에 영향을 주는 독립변수가 복수일 때 분석하는 방식
  • 다중회귀분석은 단일회귀분석에서 독립변수1 + 독립변수2 + 독립변수3 + ,,,, 하는 방식.
  • lm() 함수 사용.

값 분석

  • 유의 수준이 8.65/10^11 = 0.0000000000865 로 유의 수준 .001 보다 작아 회귀 모형에 적합.
  • disp는 유의 수준 조건을 넘고, hp/wt는 혀용치 안이기에 영향력이 있음.
  • Adjusted R-squared: 0.8083 : 결정 계수는 1에 가깝기에 회귀 모델 설명력이 높음.
  • 결론 : 회귀 모형 p < .001에서 적합 하며, 3개의 독립 변수가 연비에 미치는 회귀 계수는 hp가 -0.03, wt가 -3.8이었고, disp는 없었다. wt가 영향력이 가장 컸다.
